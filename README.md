# Ray-cast-1

This was my 1st toy raycaster made in apr-sep 2021. Then, I've revised the code.

## Dependencies

SDL2 and SDL2 mixer

## Build

it has a Makefile, so:

```sh
make
```

## Usage

Move with W-A-S-D keys. Quit with Q.
